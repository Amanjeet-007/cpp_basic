# 🏦 Bank Management System

A simple **Bank Management System** project that allows users to create bank accounts and perform common banking operations such as deposit, withdrawal, transaction history tracking, and viewing account balances.

This project demonstrates fundamental programming concepts such as **data management, user input handling, and basic banking operations**.

---

# 📌 Features

The system provides the following functionalities:

### 1️⃣ Create Account

Users can create a new bank account by providing:

* **Name**
* **Account Number**
* **Initial Deposit**

Once the account is created, it is registered in the system and can be used for further operations.

---

# ⚙️ Available Operations

After account registration, users can perform the following banking operations:

### 💰 1. Deposit Amount

Users can deposit money into their account.

Required input:

* Account Number

---

### 💸 2. Withdraw Amount

Users can withdraw money from their account.

Required input:

* Account Number

The system verifies the balance before processing the withdrawal.

---

### 📜 3. Transaction History

Users can view all previous transactions related to their account.

Required input:

* Account Number

The system displays:

* Deposits
* Withdrawals
* Account activities

---

### 💳 4. View Balance

Users can check their current account balance.

Required input:

* Account Number

---

### 📋 5. List of Registered Accounts

Displays all accounts currently registered in the system with their details.

Example details shown:

* Account Holder Name
* Account Number
* Current Balance

---

### 🚪 6. Exit

Users can safely exit the program when they are done using the system.

Input:

```
0
```

---

# 🔄 System Workflow

```
Start Program

 ├── Deposit Amount
```
